Interface web moderna e responsiva para controle de dispositivos IoT via MQTT com ESP32.

🚀 Abrir Dashboard 📖 Documentação

🎛️ Controle em Tempo Real

Controle lâmpadas, tomadas, irrigação e outros dispositivos instantaneamente via MQTT.

📊 Monitoramento

Gráficos em tempo real de temperatura e umidade com dados do sensor DHT22.

🌓 Dark/Light Mode

Alternância entre tema escuro e claro com persistência de preferências.

📱 Responsivo

Interface adaptável para desktop, tablet e smartphone usando Bootstrap 5.

⚡ Animações

Efeitos visuais suaves e indicadores de status com animações CSS modernas.

💾 Persistência

Estados dos dispositivos e configurações salvos no localStorage do navegador.

🛠️ Tecnologias Utilizadas

HTML5 CSS3 JavaScript ES6+ Bootstrap 5 Chart.js Paho MQTT Font Awesome WebSocket

🔧 Configuração MQTT

Para conectar com seu ESP32, configure o broker MQTT no dashboard:

  • Broker: broker.hivemq.com (padrão)
  • Porta: 8000 (WebSocket)
  • Protocolo: ws:// ou wss://
